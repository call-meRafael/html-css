Principais formatações

Negrito

Texto em negrito pode ser feito com a tag <strong> ou <b>.

Itálico

Texto em itálico pode ser feito com a tag <em> ou <i>

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

Também podemos aplicar uma cor diferente a esta marcação

Texto marcado com cor diferente

Texto riscado

Texto riscado pode ser usado com a tag <del>

Subscrito e sobrescrito

Podemos usar a tag <sub> para texto subscrito e <sup> para texto sobrescrito
